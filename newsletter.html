Happy Fall, Airbraker!

As I’m sure many of you have seen, we have some big announcements for this edition of the newsletter. The biggest announcement (by far) is the release of Airbrake 2.0, which includes a complete re-architecture, new integrations, a new UI, completely new Javascript notifier, and an alpha release of our version 3 API.

If you’ve logged in to Airbrake recently, you’ve probably noticed it looks different. Really different. We’ve redone our homepage, added a new dashboard with simpler side navigation, and made our integrations with other developer tools much simpler. We’ve also improved the signup process, and allowed users to select which language they’re using with our many notifier versions.

We’ve rebuilt our backend and are now running on Rackspace’s robust infrastructure after migrating over 1.6 billion files. We’ve also re-architected in Go, and can now handle up to 30,000 requests per second. This brings a whole new level of stability and reliability to our customers.

Lastly, we’ve released an alpha v3 API. This alpha API supports JSON, multi-threaded backtraces, POST and much better performance over V2. We’ll be adding more comprehensive documentation as this notifier API migrates out of beta.


Updated Javascript notifier.

We’ve also rebuilt our Javascript notifier from the ground up. Our new JS notifier contains features not found in any other client-side reporters, including full Unicode and source-mapping support. These features allow developers to receive context-rich reports from a huge range of browsers. The notifier has also been rebuilt for performance, keeping load times to a minimum via non-blocking loading and an asynchronous API.
